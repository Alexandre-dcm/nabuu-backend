# Nabuu

## Description

Nabuu is an online platform for creating documentations, articles and tutorials

## Installation

```bash
$ npm install
```

## Run migrations

```bash
$ npm run prisma:dev:deploy
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
